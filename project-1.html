Buscador anónimo

Crear una aplicación para realizar busquedas anónimas. Es una aplicación para realizar busquedas en Google, utilizando una IP diferente en cada busqueda, por medio de proxies.

Este enfoque es utilizado en el famoso startpage.com, donde ellos actuan como un proxy para buscar en Google; considero este enfoque más privado que usar el mismo DuckDuckGo, pues para usar DuckDuckgo tienes que confiar en ellos. ¿Y si mejor no confiamos en nadie? Es por eso que se propuso esta alternativa

Utilizar startpage.com se hará notar directamente que el que busca está buscando a travez de startpage.com, mejor haber buscado el mismo en Google.com con Tor o VPN; sin embargo, Google no permite el uso de Tor para realizar busquedas

Otra solución sería que uses un VPN y busques directamente en Google. Sería la segunda mejor solución, si esa es tu solución, consulta para utilizar el navegador de Tor Browser sin usar la red Tor, de esta forma protegerte del fingerprinting que implica utilizar tu propio navegador Web:

    La Aplicación sigue este esquema: un Cliente que por medio de VPN se conecta a una proxie aleatorea de entre miles y miles disponibles.

    Cliente << VPN << Proxie

    En esta aplicación usaras el VPN de tu preferencia, creo que la clave para que esto funcione está en que cada quien tenga la opción de elegir sus mejores opciones, y lo más importante de este proyecto será la forma de recopilar proxies, basta con que sean de distintas fuentes. Si una proxie es comprometida no tendría el mayor problema porque primero eres anónimo para la proxie, y segundo porque solo se habría quedado con una sóla búsqueda
