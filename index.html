El acortador de links más privado del mundo

1. El sitio funciona con github, por lo que el uso que hagas del sitio permanece en secreto para todos, incluyendo a los creadores del sitio (@princessgentoo). La información del uso del sitio la usa github para mejorar sus servicios y asegura que esa información no se usa para identificar a nadie.

2. Servicios como Cloudflare invalidan el punto número uno, y permite que incluso los dueños del sitio web puedan obtener información sobre los visitantes tales como si ip, su país y navegador web; por esta razón se evitó el uso de CDNs.

3. Es el primer acortador de links 100% transparente, pues los sitios con github muestran públicamente el código con el que funciona todo el sitio: De esta forma se puede descargar el sitio entero ejecutando:
git clone github.com/gentooproject/gentooproject.github.git
