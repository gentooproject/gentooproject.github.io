Crear un sistema de desanonimización de acortadores de links:

Nota: Este es un proyecto hipotético, comprender el funcionamiento de Tor y la red Bitcoin requiere profundos conocimientos en informática

Consiste crear una base de datos como la de bitcoin donde los links de bit.ly goo.gl y otros sean almacenados en un sistema similar al P2P, y que se pueda acceder a esta base de datos de forma anónima utilizando Tor o una tecnología similar a Tor

Esta base de datos de hará por medio de dos formas, un bot analizará constantemente el internet en busqueda de links acortados, y la otra es el proceso en el que se añaden manualmente

Método de codificación:

  1. Se genera un hash del link y se envia al servidor para guardarlo
  2. Se descargan 3 llaves públicas de tres servers de verificación, y acontinuación se envía a ellos encriptado el link para su procesamiento.
  3. A continuación se recibe la url encriptada de cada uno de los servers de verificación que debe ser identica en los 3 servers(lo encriptaron con nuestra llave publica temporal).
  4. Con la extensión se encripta la URL larga utilizando la url acortada como contraseña y se envía al servidor el encriptado resultante.
