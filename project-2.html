Crear un sistema de desanonimización de acortadores de links:

Consiste crear una base de datos como la de bitcoin donde los links de bit.ly goo.gl y otros sean almacenados en un sistema similar al P2P, y que se pueda acceder a esta base de datos de forma anónima utilizando Tor o una tecnología similar a Tor

Esta base de datos de hará por medio de dos formas, un bot analizará constantemente el internet en busqueda de links acortados, y la otra es el proceso en el que se añaden manualmente

Método de codificación:

  1. Se genera un hash del link y se envia al servidor para guardarlo
  2. Se descargan 3 llaves públicas de tres servers de verificación, y acontinuación se envía a ellos encriptado el link para su procesamiento. Se optiene un hash, el cual se comprueba que sea idéntico, verificación que hará el primer server del paso uno. Y este hash es enviado como respuesta hasta tu cliente por medio del servidor para hacer también la validación en tu propia computadora.
  3. A continuación se recibe la url encriptada de uno de los servers de verificación (lo encriptaron con nuestra llave publica temporal) y se comprueba su integridad a travez del hash recibido del servidor.
  5. Con la extensión se encripta la URL larga utilizando la url acortada como contraseña y se envía al servidor el encriptado resultante.
