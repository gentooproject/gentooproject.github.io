Thanks to the curl command, you can pass code direcly from a website into the bash shell

In order to watch the code , you can use:

curl + https://rpm.li/c/name

Notes:

  • Please do not forget to add https at the beggining because it protects the code.
  • If you use your web browser for watching the code, it will be downloaded as a binary file, because it's intended to be a command line utility

CLI utility installation

The cli utility allows to run these commands pretty strightforward, GPG support will be added later, so you can be certain that each command is not modified nor altered.

bash <(curl https://rpm.li/c/install)
