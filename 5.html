Navegar en un internet seguro

Estas ideas están basadas en el uso de Firefox, para protección más intensa es recomendado lo siguiente:

  1. Poder conectarse por medio de un túnel SSH a un servidor
  2. Un Firewall dedicado que limite el tráfico
  3. Qubes Os o una máquina virtual, en el caso de elegir Qubes OS, Qubes OS ya incluye una máquina virtual que actua como Firewall

      4. Al firewall entonces se le aplicará una política de denegar absolutamente todas las ips y puertos, en exepción de una sóla ip que es a donde se habrá de conectar a un túnel SSH. De esta forma, se limita todo a este punte SSH y limita a muchos tipos de ataque que se conectan a servers remotos.

      1. Bloquear Javascript y Iframes

      Esto lo puedes conseguir instalando:

      https://addons.mozilla.org/en-US/firefox/addon/noscript

      2. Instala Ublock origin

      Esto nos va a ayudar a bloquear la publicidad y algunos rastreadores ocultos que sólo te exponen

      https://addons.mozilla.org/en-US/firefox/addon/ublock-origin

      3. Instala Certificate Patrol

      Este es uno de los puntos más vulnerables al navegar por internet, un certificado SSL puede ser falseado sin mucha dificultad, pero para esto existe Certificate Patrol, que nos ayuda a combrobar que el certificado es exactamente el de siempre.

      4. Crear una lista de sitios buenos, y fuera de esa lista ningún sitio web puede cargarse

      Este es el punto más paranoico, aunque no se si paranoico sea una buena palabra ya que los hackers cada vez desarrollan nuevas técnicas más avanzadas.
