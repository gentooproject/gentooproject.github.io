Use Tor Browser using normal Internet in Linux

The procedure is almost the same than in Windows. The difference is that you need to delete some strings

  • Install FoxyProxy
  • Disable the extension "Tor Browser Launcher"
  • type about:config
  • Toggle network.proxy.socks_remote_dns to false
  • remove the text in network.proxy.socks
  • remove the value in network.proxy.socks_port
Source

This is part of a function called Transparent Proxy, in that file is listed all the modifications done here, this method is alternative and manual.

Usar Tor browser con tu internet normal en Linux

Los pasos son casi identificos que los de Windows. La diferencia es que vamos a borrar de la configuración unos valores:

  • Installar FoxyProxy
  • Desactivar la extension de launch Tor Browser
  • entrar a about:config
  • cambiar network.proxy.socks_remote_dns a false
  • borrar el contenido de network.proxy.socks
  • borrar el contenido de network.proxy.socks_port
Referencia

Esto que hicimos es parte de una función llamada Proxy Transparente, y viene listado ahí todos los cambios que hicimos si se habilita esta variable, este es un método alternativo y manual.
